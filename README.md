## Задача 1

Все натуральные числа покрашены в некоторые цвета. Известно, что для любых неотрицательных целых $m,n$ числа вида

$$m + n,$$
$$m ^ 2 + n^2 + 1$$

покрашены в один цвет. Все ли числа покрашены в один цвет?

Ваша задача:

1. Проверить (на python, см. приложенные файлы), что все числа, не превосходящие наперед заданного $N=10$ покрашены в один цвет;

2. Доказать или опровергнуть, что все числа покрашены в один цвет.

### Что нужно сделать?

1. Реализовать методы класса Checker из numbers_checker.py (в особенности get_path);

2. Доказательство затехать.

- Общий ИПЦ (корень)

Данные для тестирования [здесь](https://github.com/AllonHammer/CPI_HRNN/blob/master/data/cpi/cpi_us_dataset.csv).

**Что предоставить:**

1. Код класса `HierarchicalLinearRegression` и пример его использования
